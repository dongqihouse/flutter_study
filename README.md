# flutter_study

## 常用Widget
1. richText
2. FittedBox
    场景： 字体自适应大小，可以控制自适应范围比如可以缩小，但是限制放大
3. safeArea
    场景: 刘海屏， 而且自带padding属性
4. LayoutBuilder
    场景： debug获取父组件约束， 或者子组件约束需要依赖父组件布局
5. DefaultTextStyle
    场景： 子组件统一样式
6. InteractiveViewer
    场景： 可交互视图，上下左右滚动放大，例如地图或者放大图片的需求
